# Admin Dashboard

This page provides a detailed overview of the Admin Dashboard in HaberNexus.

## 1. Overview

The Admin Dashboard is a protected area of the application that is only accessible to users with the `ADMIN` role. It provides a central location for managing all aspects of the system.

## 2. Features

| Feature | Description |
|---|---|
| **RSS Feed Management** | Add, edit, and delete RSS feeds. |
| **User Management** | View, edit, and delete users. |
| **System Settings** | Configure system-wide settings, such as the cron schedule and AI model parameters. |

## 3. Technologies Used

| Technology | Purpose |
|---|---|
| **Next.js** | Powers the Admin Dashboard UI. |
| **Prisma** | Provides the interface to the SQLite database for managing data. |

## 4. Future Enhancements

- **Analytics:** Display detailed analytics about the system, such as the number of articles generated, the number of users, and the most popular RSS feeds.
- **System Health Panel:** Display real-time information about the health of the system, such as the status of the cron job and the AI model.
