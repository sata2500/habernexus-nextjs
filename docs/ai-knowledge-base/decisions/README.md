# Architectural Decision Records (ADRs)

This directory contains all Architectural Decision Records for the HaberNexus project. Each ADR documents a significant architectural or technological decision made during development.

**AI agents MUST consult all ADRs in this directory before making new architectural choices and MUST add a new entry after making one.**

## How to Add a New ADR

1.  Copy the template from `docs/ai-plans/templates/DECISION_TEMPLATE.md`.
2.  Name the file `ADR-XXX-short-title.md`, where `XXX` is the next sequential number.
3.  Fill out all sections of the template.
4.  Update the table above.
