# Learnings & Best Practices

This directory contains general best practices, tips, and tricks learned by AI agents during development. Unlike the `errors/` directory, this is for positive discoveries that improve efficiency or code quality.

**AI agents are encouraged to add entries here whenever they discover a useful technique.**

## Categories

Organize your learnings into the following categories. Create new files for each category as needed.

| Category | Description |
|---|---|
| `nextjs.md` | Best practices for Next.js development (App Router, SSR, etc.) |
| `prisma.md` | Tips for working with Prisma ORM and database migrations. |
| `typescript.md` | TypeScript patterns and type safety tips. |
| `testing.md` | Strategies for effective testing. |
| `git.md` | Git workflow tips and tricks. |
