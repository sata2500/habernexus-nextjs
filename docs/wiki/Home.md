# HaberNexus Wiki

**HaberNexus**, yapay zeka destekli, tam otomatik bir haber agregasyon platformudur. Bu wiki, projenin kurulumu, yapılandırması ve kullanımı hakkında kapsamlı bilgiler içerir.

## Hızlı Başlangıç

Tek komutla kurulum için:

```bash
wget -qO- https://raw.githubusercontent.com/sata2500/habernexus-nextjs/master/scripts/install.sh | bash
```
